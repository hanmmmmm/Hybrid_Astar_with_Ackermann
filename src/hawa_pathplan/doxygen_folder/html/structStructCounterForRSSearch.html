Detailed Description

This struct handles the logic for triggering the RS curve searching, during the the hybrid astar exploration. The value of interval would be modified to the actual desired value. Briefly, everytime when counter is an interger multiplication of the interval, then the hybrid astar should try to see if any RS curve connecting to the goal pose.
