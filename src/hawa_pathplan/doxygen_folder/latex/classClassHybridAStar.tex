\doxysubsection{Detailed Description}
This file is the implementation of the Hybrid Astar searching algorithm on an occupancy grid map. This implementation contains only the incremental searching (the part that\textquotesingle{}s like original a-\/star) and the Reeds\+Shepp curve searching. This does not have the parts like Voronoi field and numerical optimization.
